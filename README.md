# Tradutor Online 🇧🇷 → 🇬🇧

Um tradutor simples, rápido e bonito feito com **HTML, CSS e JavaScript puro**, consumindo a API pública MyMemory.

**[Acessar o Tradutor →](https://vitor213.github.io/Tradutor/)**

## ✨ Funcionalidades

- Tradução em tempo real do **Português para Inglês**
- Interface moderna com **Bootstrap 5**
- Validação do texto (remove caracteres especiais)
- Feedback visual durante a tradução
- Tratamento de erros e estados vazios
- Totalmente funcional no navegador (client-side)

## 🛠️ Tecnologias Utilizadas

- **HTML5** + **CSS3**
- **JavaScript Vanilla** (Async/Await, Fetch API)
- **Bootstrap 5**
- [MyMemory Translation API](https://mymemory.translated.net/)
